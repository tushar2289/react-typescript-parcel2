# Overview

This is a bootstrap project for a React webapp using the following:

1. Typescript
2. Eslint
3. Prettier
4. Husky
5. Lint staged
6. Jest
7. React testing library

# Scripts

## Run dev server

```
yarn dev
```

## Build for production

```
yarn build
```

## Run tests

```
yarn test
```
